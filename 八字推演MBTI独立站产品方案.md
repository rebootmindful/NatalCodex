## 1. 产品定位与核心价值 (Product Positioning & Core Value)

### 1.0 品牌定义 (Brand Identity)

**产品名称：** NatalCodex

**名称释义：**

* **Natal**：拉丁语“出生的、诞生的”，直指产品核心——基于出生时刻的分析
* **Codex**：拉丁语“法典、代码集、古代手抄本”，象征着系统化的知识体系
* **整体含义：**“关于出生的法典”或“生命诞生的源代码”，完美融合Cyber-Zen（赛博禅意）风格

**核心产品：Soulprint Card（灵魂指纹卡）**

用户通过输入出生日期，生成一张可视化的个性化卡片，呈现：

* 八字命盘结构化解析
* MBTI性格类型推演结果
* AI生成的深度个性化分析（付费版）
* 可分享的视觉设计

**品牌口号 (Slogan)：**

> **"Unlock the Source Code of Your Soul"**（解锁你灵魂的源代码）

**品牌调性：**

* **Cyber-Zen**：东方古老智慧 × 未来科技美学
* **复古未来主义**：Retro-Futurism，古代手抄本 × 数字代码
* **科学化包装**：强调分析、解码、蓝图，而非算命或宿命

### 1.1 独特卖点 (USP)

**"Ancient Wisdom Meets Modern Psychology"（古老智慧遇上现代心理学）**

本产品的核心USP不在于简单的算命，而在于提供一种\*\*"非问卷式"（Non-questionnaire based）\*\*的人格探索路径。

* **零偏差输入**：传统的MBTI测试依赖用户的主观自评，容易受到"社会期许效应"（Social Desirability Bias）影响。而本产品基于出生时间（客观数据），提供了一种"先天出厂设置"的解读视角。
* **跨时空的自我确认**：当几千年前的东方算法准确描述了用户当下的性格特质时，会产生极强的"Aha Moment"（顿悟感）和神秘的连接感。

### 1.2 为什么外国用户会对"八字推MBTI"感兴趣？

* **Astrology 2.0 的需求**：西方用户对星座（Astrology）、塔罗（Tarot）和人类图（Human Design）有极高的接受度。八字（Four Pillars）可以被重新包装为"更精准、更个性化的东方星盘"。
* **对MBTI的疲劳感**：MBTI非常流行，但填写100道题很累。用户渴望新奇的、低门槛的替代方案。
* **东方神秘主义的吸引力**：Zen（禅）、Taoism（道）在欧美有深厚的文化滤镜，象征着深邃的智慧和内心的平静。

### 1.3 目标市场定位

* **主要市场**：北美（USA/Canada）、欧洲（UK/Germany/France）、东南亚（Singapore/Malaysia，作为早期种子用户验证区）。
* **利基市场**：身心灵（Spirituality）爱好者社区、心理学爱好者、自我提升（Self-help）寻找者。

绝对不能使用 "Fortune Telling"（算命）或 "Superstition"（迷信）这类词汇。

* **核心类比**：**"The Energetic DNA at Your Birth"（你出生时的能量DNA）**。
* **解释逻辑**：
  * **八字 (BaZi)** → **"The Four Pillars of Energy"** (Year, Month, Day, Hour).
  * **五行 (Five Elements)** → **"The Five Elemental Forces"** (Wood, Fire, Earth, Metal, Water) that shape your psychological structure.
  * **命盘** → **"Natal Energy Chart"**。

为了避免被视为"伪科学"（Pseudoscience），需要采用\*\*"荣格心理学+统计学"\*\*的包装路径。

### 5.4 AI API 集成：Vertex AI 协议（非流式）

**技术选型**：接入遵循Google Vertex AI标准协议的第三方AI服务。

**交互模式：非流式 (Non-Streaming)**

采用**非流式**模式，等待AI模型生成完整回复后，一次性返回给前端。

**优缺点分析：**

✅ **优点**：

* **数据完整性**：便于后端JSON解析、格式校验、敏感词过滤
* **开发简单**：无需处理复杂的SSE连接和状态管理
* **适合批处理**：后台自动生成报告的理想场景
* **成本可控**：单次请求即单次计费，无长连接开销

⚠️ **缺点**：

* **首字延迟高**：用户需等待完整生成过程，感知延迟较明显
* **解决方案**：配合优秀的Loading动画（如“星盘转动”、“能量汇聚”）营造仪式感

**代码示例（Vertex AI 标准协议）：**

```typescript
// app/api/generate-ai-report/route.ts
import { GoogleAuth } from 'google-auth-library';

const PROJECT_ID = process.env.GOOGLE_PROJECT_ID;
const LOCATION = 'us-central1';
const MODEL_ID = 'gemini-1.5-pro'; // 或其他兼容模型
const API_ENDPOINT = `https://${LOCATION}-aiplatform.googleapis.com/v1/projects/${PROJECT_ID}/locations/${LOCATION}/publishers/google/models/${MODEL_ID}:predict`;

/**
 * 非流式调用 AI 模型
 */
export async function POST(req: Request) {
  const { baziData, mbtiResult } = await req.json();
  
  // 1. 验证支付状态（确保用户已付费）
  // await verifyPayment(req);
  
  // 2. 构建 Prompt
  const prompt = `
Role: You are "Eastern Echo", a wise sage bridging Ancient Chinese Philosophy (BaZi) and Modern Psychology.

User Data:
- Day Master: ${baziData.dayMaster}
- MBTI: ${mbtiResult}
- Dominant Element: ${baziData.strongestElement}

Generate a 2000-word soul analysis with career guidance...
  `;
  
  // 3. Google Auth
  const auth = new GoogleAuth({
    scopes: 'https://www.googleapis.com/auth/cloud-platform'
  });
  const client = await auth.getClient();
  const accessToken = await client.getAccessToken();
  
  // 4. 调用 Vertex AI（非流式）
  const requestBody = {
    instances: [{ content: prompt }],
    parameters: {
      temperature: 0.7,
      maxOutputTokens: 2048,
      topP: 0.9
    }
  };
  
  try {
    const response = await client.request({
      url: API_ENDPOINT,
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${accessToken.token}`,
        'Content-Type': 'application/json'
      },
      data: requestBody
    });
    
    // 5. 解析响应
    const generatedText = response.data.predictions[0].content;
    
    return Response.json({ 
      success: true,
      report: generatedText 
    });
  } catch (error) {
    console.error('Vertex AI Error:', error);
    return Response.json({ 
      success: false, 
      error: 'AI generation failed' 
    }, { status: 500 });
  }
}
```

**T/F (理智/情感) 维度：**

* **七杀/正官/正印强**（权威、逻辑）→ **T**
* **食神/伤官强**（表达、情绪）→ **F**

**J/P (判断/感知) 维度：**

* **格局稳固、五行流通**→ **J**

## 8. 风险与挑战 (Risks & Challenges)

### 8.1 文化误解风险

**问题：**

部分西方用户可能将八字与“水晶球算命”混为一谈，认为产品是“迷信”或“伪科学”。

**应对策略：**

✅ **强调心理学框架**：在所有文案中强调这是“Archetypal Psychology”（原型心理学）工具。

✅ **引用荣格：Carl Jung**：在 About 页面明确写："Carl Jung, the founder of analytical psychology, deeply studied the I Ching and Eastern philosophy."

✅ **Disclaimer 声明**：Footer 加上：

> "This tool is for entertainment and self-reflection purposes. It should not replace professional psychological or medical advice."

> ✅ **避免绝对化语言**：不说 "You WILL fail"，而说 "You may find challenges in..."

### 8.2 准确性质疑的应对

**问题：**

用户可能会说：“这也太不准了！我明明不是这样的人！”

**应对策略：**

✅ **Barnum Effect 缓冲**：在报告中加入：

> "Remember: This analysis reflects your *innate tendencies*, not your current identity. Life experiences shape us differently."

> ✅ **反馈机制**：在结果页加 "How accurate was this? \[1-5 stars]"，收集数据优化算法。

> ✅ **多维度分析**：不要只给一个 MBTI 类型，而是给出 Top 3 可能性 + 每个维度的百分比分数：

* E/I: 65% Introverted
* S/N: 55% Intuitive (边界人格)
* T/F: 70% Feeling
* J/P: 60% Perceiving
