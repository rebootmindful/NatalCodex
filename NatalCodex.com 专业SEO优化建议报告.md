## 🔧 六、技术SEO优化建议

### 6.1 页面速度优化

**当前问题（需验证）**：

* 图片资源可能未充分优化（查看 mandala.webp.png 等）
* 可能缺少 CDN 加速

**优化建议**：

1. ✅ 图片格式优化：
   * 使用 WebP 格式（已部分采用）
   * 实施响应式图片（srcset）
   * 懒加载（lazy loading）
2. ✅ 代码优化：
   * JavaScript 拆分与按需加载
   * CSS 关键渲染路径优化
   * 移除未使用的代码
3. ✅ CDN部署：
   * 使用 Cloudflare 或 AWS CloudFront
   * 静态资源缓存策略

**目标指标**：

* First Contentful Paint (FCP) < 1.8s
* Largest Contentful Paint (LCP) < 2.5s
* Cumulative Layout Shift (CLS) < 0.1

### 6.2 移动端优化

**检查清单**：

* ✅ 响应式设计已实现
* ✅ 移动端可用性测试（Google Mobile-Friendly Test）
* ✅ 触摸元素大小符合标准（至少 48x48 像素）
* ✅ 避免使用插件内容

### 6.3 结构化数据标记

**推荐实施的 Schema.org 类型**：

1. **Organization Schema**（首页）

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "NatalCodex",
  "description": "融合古老东方智慧与现代心理学，发现你的人格蓝图",
  "url": "https://natalcodex.com",
  "logo": "https://natalcodex.com/logo.png"
}
```

1. **WebApplication Schema**（工具页）

```json
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "八字命盘生成器",
  "applicationCategory": "LifestyleApplication",
  "offers": {
    "@type": "Offer",
    "price": "0",
    "priceCurrency": "USD"
  }
}
```

1. **FAQPage Schema**（常见问题）

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "什么是八字命理？",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "八字（BaZi）是中国古老的命理学系统..."
    }
  }]
}
```

* 问题：Google/百度算法变化可能影响排名
